# AI/ML Portfolio --- Professional UI/UX Design Flow

## 1. Design Philosophy

**Objective:**\
Create a clean, modern, and senior-level portfolio that communicates
credibility, clarity, and technical depth.

**Core Principles:** - Minimal color system (1 primary accent +
neutrals) - Structured horizontal layout - Controlled glassmorphism
usage - Strong typography hierarchy - Data-forward presentation - High
whitespace discipline

------------------------------------------------------------------------

## 2. Visual Identity System

### Color System

**Background (Light Mode)** - Base: #F8FAFC - Surface: #FFFFFF - Subtle
Layer: #F1F5F9

**Background (Dark Mode - Optional)** - Base: #0F1115 - Surface:
#161A23 - Subtle Layer: #1C2230

**Primary Accent** - Indigo: #6366F1

**Text** - Primary: #111827 (light) / #F9FAFB (dark) - Secondary:
#6B7280 - Muted: #9CA3AF

Avoid multi-color gradients across the page. Use gradients only in CTAs
or subtle highlights.

------------------------------------------------------------------------

## 3. Typography System

**Font Style** - Modern sans-serif (e.g., Inter / SF Pro / Geist)

**Hierarchy** - H1: 64--72px \| Bold \| Tight line height - H2: 32--40px
\| Semibold - Body: 18--20px \| Regular \| 1.6 line height - Small
labels: 12--14px \| Medium \| Uppercase with tracking

Avoid coloring inline text. Use weight instead of color for emphasis.

------------------------------------------------------------------------

## 4. Layout Grid System

Use a 12-column grid system.

**Container Width** - Max width: 1200--1280px - Horizontal padding:
24--32px

Maintain consistent vertical spacing (80--120px between sections).

------------------------------------------------------------------------

## 5. Hero Section (Above the Fold)

### Layout: 60 / 40 Split

**Left Column (Primary Messaging)** - Headline (2 lines max) - Short
value proposition (1--2 sentences) - Primary CTA - Secondary CTA

**Right Column (Glass Metrics Panel)** - 2x2 metric grid - "Available
for work" indicator - Optional subtle line chart or abstract visual

### Glass Panel Specifications

-   Background: rgba(255,255,255,0.06)
-   Border: 1px solid rgba(255,255,255,0.12)
-   Backdrop blur: 16--24px
-   Shadow: 0 8px 32px rgba(0,0,0,0.15)

Do not apply glass globally. Limit to hero panel and select cards.

------------------------------------------------------------------------

## 6. Metrics Design (Side-by-Side)

Avoid vertical stacking.

**Option A: Inline Metrics Row** 3+ Years \| 20+ Projects \| 15+
Deployments

Separated by subtle dividers.

**Option B: Dashboard Card** 2x2 grid layout inside glass container.

Large numeric emphasis (36--40px). Small uppercase label beneath number.

------------------------------------------------------------------------

## 7. Projects Section

### Layout: 2--3 Cards Per Row

Each project card: - Project title - Short outcome-focused description -
Tech stack tags - Measurable impact metric - Link or preview CTA

Card Styling: - Subtle surface background - Soft shadow - 12--16px
border radius - Hover: slight elevation (translateY -4px)

------------------------------------------------------------------------

## 8. Experience + Tech Stack Section

Two-column layout.

**Left: Experience Timeline** - Role - Company - Duration - Impact
bullet points

**Right: Skills Visualization** - Categorized tech stacks - Grid
format - Avoid excessive logos

------------------------------------------------------------------------

## 9. Contact Section

Horizontal layout strip.

Include: - Short call-to-action statement - Email / LinkedIn / GitHub -
Optional booking link

Use accent button as final primary action.

------------------------------------------------------------------------

## 10. Navigation Design

-   Minimal glass navbar
-   No excessive icons
-   Active state = subtle underline or pill
-   Compact spacing

Navbar should feel lighter than hero section.

------------------------------------------------------------------------

## 11. Micro-Interactions

-   200--300ms ease transitions
-   Subtle hover elevation
-   Gentle fade-in stagger for sections
-   No aggressive motion or bounce effects

Maintain professional restraint.

------------------------------------------------------------------------

## 12. What to Avoid

-   Heavy gradients across full background
-   Grid overlays or decorative noise
-   Excess glow effects
-   Too many glass layers
-   Overuse of purple accents
-   Long vertical stacked content

------------------------------------------------------------------------

## 13. Advanced Creative Direction Options

### Option 3: Soft Neo-Glass

-   Light pastel background
-   Frosted layered panels
-   Soft depth and shadow system

------------------------------------------------------------------------

## 14. Final UX Goal

The portfolio should: - Communicate senior-level clarity - Emphasize
measurable impact - Guide the eye horizontally first - Maintain visual
calmness - Feel structured, intentional, and technically confident
